# 404s to Mars Hackathon – Cow Abduction 404

Este repositorio contiene el proyecto desarrollado por el equipo **COW**: Valeria Chacín, Anahí Kolofon, Micaela Arce, Virginia Sarrais y Sofía Ruiz, para el **Hackathon UDIT**, dentro del desafío **“404s to Mars”**.  
El objetivo es transformar la tradicional página de error 404 en una experiencia web creativa y humorística: una nave espacial que abduce una vaca en una escena nocturna.

## Propósito y Audiencia

- **Para Usuarios:** Ofrecer una página 404 entretenida y visualmente atractiva que convierta un error técnico en una experiencia memorable.  
- **Para Instructores y Jurado:** Demostrar el trabajo en equipo, el diseño visual y las habilidades en animación con CSS dentro del tiempo limitado del hackathon.

## Concepto Creativo

**Título:** Cow Abduction 404  
**Historia:**  
El usuario aterriza en una página inexistente. En la oscuridad de la noche, aparece una nave espacial que abduce a una vaca confundida.  
Mensaje mostrado: “404: Página abducida. Parece que los alienígenas se la han llevado.”  

**Mensaje Principal:**  
404 – Página Abducida  

**Llamada a la Acción:**  
"Beam me back home" – botón que redirige al sitio principal.

## Tecnologías Utilizadas

- **HTML5:** Estructura base de la página.  
- **CSS3:** Estilos y animaciones (sin frameworks).  
- **JavaScript opcional:** Uso mínimo, solo si se requiere para efectos simples.  
- **GitHub Pages:** Alojamiento y despliegue en línea.
